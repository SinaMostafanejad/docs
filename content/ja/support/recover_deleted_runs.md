削除された run を復元するには、次の手順を完了します。

- Project Overview ページに移動します。
- 右上隅の三点ドットをクリックします。
- **Undelete recently deleted runs** を選択します。

**注意**:
- 過去 7 日以内に削除された run のみ復元できます。
- 復元がオプションでない場合、W&B API を使用してログを手動でアップロードできます。
