W&B クライアントライブラリは Python 3.5 をバージョン 0.11 までサポートしていました。Python 3.5 のサポートはその終末と一致するバージョン 0.12 で終了しました。詳細については、[version 0.12 release notes](https://github.com/wandb/wandb/releases/tag/v0.12.0) をご覧ください。
