W&B Sweep が開始されると、Sweep configuration を変更することはできません。しかし、任意のテーブルビューに移動し、チェックボックスを使用して runs を選択し、**Create sweep** メニューオプションを選択することで、過去の runs に基づいた新しい Sweep configuration を生成できます。
