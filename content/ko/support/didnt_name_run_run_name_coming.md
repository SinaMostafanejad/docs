만약 run에 명시적으로 이름이 지정되지 않은 경우, W&B는 프로젝트에서 해당 run을 식별하기 위해 임의의 이름을 할당합니다. 임의 이름의 예로는 `pleasant-flower-4` 및 `misunderstood-glade-2`가 있습니다.
