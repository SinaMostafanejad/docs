W&B 클라이언트 라이브러리는 0.11 버전까지 Python 3.5를 지원했습니다. Python 3.5 지원은 수명 종료에 맞춰 0.12 버전에서 종료되었습니다. 자세한 내용은 [version 0.12 release notes](https://github.com/wandb/wandb/releases/tag/v0.12.0)를 참조하세요.
